# LE.sys | Leah Estrella's Portfolio

A high-performance, responsive engineering portfolio built with native HTML, CSS, and modern edge components. 
It features a full interactive CLI (Command Line Interface), modal window data fetching, and native system themes.

## Architecture & Stack
- **Frontend Core:** Vanilla HTML5, CSS3, JavaScript (ES6)
- **UI Library:** Lucide Icons
- **Animation System:** Custom CSS Keyframes / DOM Intersections
- **Deployment:** Ready for GitHub Pages / Vercel Edge

## Build Philosophy
This portfolio drops bloated node module installations and complex framework rendering layers in favor of lightweight, blazing-fast native script executions—bridging bare-metal control with high-end aesthetic design boundaries.

### Local Development
To launch the portfolio locally for staging:
```bash
python -m http.server 8000
```
Navigate to `localhost:8000`.
